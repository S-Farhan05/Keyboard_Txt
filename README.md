# Keyboard Reaction Lab

A simple keyboard reaction time experiment that measures how quickly users can respond to visual stimuli.

## How It Works

1. Press **SPACE** to start a trial
2. Wait for the stimulus (either **A** or **L**)
3. Press the correct key as quickly as possible
4. View your stats: trials completed, correct responses, and average reaction time

## Usage

Open `index.html` in a web browser to start the experiment.

## Technical Details

- Pure HTML/CSS/JavaScript (no dependencies)
- Measures reaction time in milliseconds
- Tracks accuracy and performance across multiple trials
